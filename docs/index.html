Developer API

Integrate escrow comparison data directly into your applications

Available Endpoints
GET/api/services

List all escrow services with current rates

GET/api/services/{id}/fees

Calculate fees for specific service and amount

GET/api/compare

Compare multiple services side-by-side

GET/api/status

Real-time service availability status

Code Example
// Calculate fees across all services
const response = await fetch('https://api.escrowcompare.io/api/services/fees', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 10000,
    type: 'real-estate',
    location: 'us'
  })
});

const fees = await response.json();
console.log(fees.services); // Array of services with calculated fees
